# The Sports Pod Chatbot

A production-ready FAQ customer service chatbot for The Sports Pod clinic.

## 🚀 Features
- **Blazing Fast AI**: Powered by Groq (`llama-3.3-70b-8192`) for near-instant responses.
- **Karpathy-Style RAG**: Pure TypeScript BM25 retrieval for intelligent context matching.
- **Streaming UI**: Tokens stream in real-time for a premium, interactive feel.
- **Self-Contained Widget**: A single `<script>` tag can embed this chatbot on any website.

## 🛠️ Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + Framer Motion
- **LLM**: Groq SDK
- **Retrieval**: Custom BM25 Implementation
- **Deployment**: Vercel
